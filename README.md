# FindAI - AI-Powered File Search

Search your files naturally: "that PDF about nginx", "video from last week", "large zip files".

## Features
- Natural language file search
- Search by content type, date range, file type, keywords
- File content indexing (text files, PDFs, documents)
- Search history with suggestions
- File preview and quick actions (open, share, delete)
- Smart filters (date, size, type)
- Fully offline - no internet required

## Tech Stack
- Kotlin 1.9.22, Compose BOM 2024.01.00
- Room 2.6.1, Material Design 3 (Dark)
- MinSdk 29, TargetSdk 34, ARM64 only
