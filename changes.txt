Key Architecture Decisions:
- MinSdk 29, TargetSdk 34, compileSdk 34, ARM64 only (ndk abiFilter)
- Room with Flow for reactive data
- NLSearchEngine parses: keywords, date refs (yesterday/last week/last month),
  file types (PDF, image, video, audio, code, archive), size hints (large/small)
- Scoring: exact_name=20, name_prefix=15, extension=12, name_contains=10, content=5
- FileIndexer scans Downloads, Documents, DCIM, Pictures, Movies, Music with depth=10
- ContentReader reads text files up to 100KB, basic PDF text extraction from streams
- DebugLogger with FA-xxx error codes (FA-001 through FA-104)
- All navigation via Jetpack Navigation Compose
- FileProvider for secure file sharing/opening
- Dark Material Design 3 with NeonBlue (#4FC3F7), NeonIndigo (#5C6BC0) accents

NOTE: gradlew binary script needs to be added via `gradle wrapper` command.
Run: gradle wrapper --gradle-version 8.5
